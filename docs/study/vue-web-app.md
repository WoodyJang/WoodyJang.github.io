# vue2-0电影web-app

## 一个使用 Vue2.0 仿豆瓣电影的项目
>github地址: https://github.com/WoodyJang/vue_movie


克隆项目到本地
>git clone https://github.com/WoodyJang/vue_movie.git

安装依赖
>npm install

在 localhost:8080 启动项目
>npm run dev
